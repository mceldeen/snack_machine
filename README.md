# Snack Machine
An implementation of a snack machine as laid out in the DDD in practice course for Pluralsight

## Setup
Must have `direnv` installed.
```sh
direnv allow
npm install
```

## Run
There's nothing interested to run yet.
```sh
pulp run
```

## Test
```sh
pulp test
```
